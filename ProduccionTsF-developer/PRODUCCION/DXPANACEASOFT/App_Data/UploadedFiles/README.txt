************************************************************
  IMPORTACIÓN DE DATOS DESDE ARCHIVO DE EXCEL
************************************************************

Para la importación de datos desde libros de cálculo de Excel, se requiere la instalación
del componente Microsoft Access Database Engine 2016, en la versión de 32 bits o 64 bits,
según corresponda.

La descarga está disponible en el siguiente enlace:
https://www.microsoft.com/en-us/download/details.aspx?id=54920
